Position

was ist Position ?

Mit der CSS-Eigenschaften Position wird gesteuert, wo die Elemente auf einer Seite dargestellt werden können

WIe funktioniert das

positionlegt fest, wie die Elemente in unserem Dokument positioniert werden Die Eigenschaften top bottom left right

position: static

Standartwert für jedes HTML Element das Element befindet sich an seiner Standartposition

static

position: relative

Das Element bleibt an seiner ursprünglichen Stelle, kann aber von dort aus mit top oder left und so weiter verschoben werden

relative

position: absolute

Das element orientiert sich am nächstgelegenen nicht-statichen Eltern Element, wenn es ein solche Element nicht gibt, orientiert es sich am HTML Element body

Best Practice: Das Eltern-Element sollte die Eigenschaft => position: relative erhalten

absolute

position: fixed

Das Element orientiert sich am Brwoserfenster es wird aus dem Dokumentfluss entfernt und bleibt beim scrollen immer an seiner position

fixed

position: sticky

Das Element scrollt mit, bis eine seiner Kante die durch top bottom left right angegebenen Koordinate erreicht hat

sticky
